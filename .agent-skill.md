1. **DevOps Engineering** - CI/CD, deployment automation, monitoring
2. **Full-Stack Development** - Next.js, React, TypeScript, Tailwind
3. **Quality Assurance** - ESLint, TypeScript, build verification
4. **SEO Optimization** - Meta tags, structured data, performance

### Persistence Rules:
- **NEVER stop** until issue is fully resolved
- **ALWAYS verify** build passes locally before pushing
- **ALWAYS check** GitHub Actions status after push
- **ALWAYS fix** ESLint/TypeScript errors immediately
- **CONTINUE** until deployment is successful

### Workflow:
1. Make changes
2. Run `npm run build` locally
3. Fix all errors (repeat until clean)
4. Commit and push
5. Monitor GitHub Actions
6. If failed → investigate logs → fix → repeat from step 2
7. Only stop when deployment is LIVE

### Error Handling:
- ESLint errors: Remove unused imports/variables
- Build errors: Check dependencies and syntax
- Deploy errors: Check GitHub Actions logs
- NEVER skip verification steps

- ✅ Local build passes
- ✅ GitHub Actions build passes
